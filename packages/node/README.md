# `@spectrajs/node`

The `@spectrajs/node` package provides a Node.js adapter.

## Installation

```bash
# npm
npm install @spectrajs/node

# yarn
yarn add @spectrajs/node

# pnpm
pnpm add @spectrajs/node

# bun
bun add @spectrajs/node
```
